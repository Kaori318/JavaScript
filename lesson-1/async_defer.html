Атрибуты async/defer – только для внешних скриптов

Специальные атрибуты async и defer используются для того, чтобы пока грузится внешний скрипт – браузер показал остальную (следующую за ним) часть страницы.

Атрибут defer

  • Поддерживается всеми браузерами, включая самые старые IE
  • Относительный порядок скриптов с defer будет сохранён
  • Всегда ждёт, пока весь HTML-документ будет готов

Атрибут async

  • Поддерживается всеми браузерами, кроме IE9
  • Выполняется по мере загрузки.
  • Не ждет готовности документа
